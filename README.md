# 🎨 酒店挂画仓储管理系统

## 快速开始

双击打开 `index.html`，即可使用完整系统。

**无需安装、无需部署、无需服务器**

---

## 功能说明

### 📍 空间管理
- 添加酒店
- 为每家酒店配置楼层、房间、挂画位置
- 每个房间可自定义 0-N 个挂画位置

### 🖼️ 画作管理
- 上传画作预览图
- 填写展签信息（作品名称、作者、年龄、阐释）
- 三级联动分配位置（酒店→楼层→房间→位置）

### 📊 全局看板
- 统计面板（酒店数、画作数、位置数、空余位置）
- 画作分布总览（卡片式展示）
- 点击画作查看详情

### 🔨 供应商工作台
- 待制作清单（已分配但未悬挂的画作）
- 查看展签信息+精确位置
- 更新状态（已装裱→已悬挂）

---

## 使用说明

### 第一步：配置空间
1. 点击左侧「空间管理」
2. 添加酒店（如：希尔顿酒店）
3. 添加楼层（如：3层、4层）
4. 添加房间（如：301、302）
5. 添加位置（如：床头背景墙、沙发上方）

### 第二步：上传画作
1. 点击「画作管理」
2. 填写展签信息
3. 上传画作预览图
4. 点击「保存画作」

### 第三步：分配位置
1. 在画作列表点击「分配位置」
2. 逐级选择 酒店→楼层→房间→位置
3. 确认分配

### 第四步：供应商制作
1. 点击「供应商工作台」
2. 查看待制作清单
3. 完成后点击「标记已装裱」→「标记已悬挂」

## 数据存储

- 所有数据保存在浏览器 LocalStorage 中
- 刷新页面数据不丢失
- 清除浏览器数据会丢失所有信息

## 后续升级路径

2. 图片存储改为阿里云 OSS
3. 添加用户认证
4. 部署到 Vercel/服务器
